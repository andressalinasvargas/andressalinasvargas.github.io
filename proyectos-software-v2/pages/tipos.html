Clasificación del software

Los proyectos de software se clasifican según la plataforma donde se ejecutan, el propósito que cumplen y el modelo de acceso que ofrecen.

🌐
Web

Sistema web

Se ejecuta en el navegador. No requiere instalación y funciona en cualquier dispositivo.

HTMLCSSJavaScriptSQL
📱
Móvil

App móvil

Instalada en smartphones. Accede a GPS, cámara y sensores del dispositivo de forma nativa.

AndroidiOSFlutter
🖥
Escritorio

App de escritorio

Se instala en el PC. Acceso completo al sistema de archivos y hardware local.

WindowsmacOSLinux
🔁
Multiplataforma

App híbrida

Una sola base de código funciona en web, móvil y escritorio simultáneamente.

FlutterElectronReact Native
⚙️
Servidor

Software de servidor

Sin interfaz gráfica. Procesa tareas automatizadas y sirve de base para otros sistemas.

Node.jsDockerLinux
🤖
Embebido

Software embebido

Diseñado para hardware específico con recursos limitados: sensores IoT y sistemas industriales.

C / C++ArduinoRaspberry Pi
