Flujo TIC — Ciclo de vida de un paquete
01 / CREAR
✓ Registro
USUARIO ingresa datos. SOFTWARE valida y asigna código único
02 / GUARDAR
✓ Base de Datos
SOFTWARE (MySQL) inserta registro. HARDWARE almacena en disco
03 / PROCESAR
✓ Logística
HARDWARE (escáner) cambia estado. RED sincroniza posición GPS
04 / COMPARTIR
✓ Entrega
USUARIO confirma recepción. RED notifica al sistema central

Componentes TIC del Sistema

🖥️
Hardware
Servidores físicos, básculas digitales, escáneres de código de barras, dispositivos GPS en camiones, computadores de los empleados, impresoras de guías.
⚙️
Software
Este sistema web (PHP), base de datos MySQL, algoritmos de enrutamiento, sistema operativo del servidor, aplicación móvil de repartidores, sistema de facturación.
🌐
Redes
Internet para acceso remoto, GPS satelital para rastreo, Wi-Fi en bodegas, red 4G/5G para dispositivos móviles, protocolo HTTPS para seguridad de datos.
👤
Usuario
Clientes que envían/reciben, empleados que operan el sistema, administradores que configuran la red, repartidores que actualizan estados desde móvil.

Tipos de Error TIC
🖥️ Error de Hardware
Disco lleno, memoria RAM agotada, dispositivo dañado, sensor defectuoso
⚙️ Error de Software
Bug en código, clave duplicada en BD, validación fallida, lógica incorrecta
🌐 Error de Red
Timeout, conexión rechazada, DNS fallido, latencia alta, paquetes perdidos
👤 Error de Usuario
Dato mal ingresado, campo vacío, formato incorrecto, acción no autorizada
